# ✨ Exciting To-Do List Web App

A fun, colourful, and animated To-Do List application built using **HTML, CSS, and Vanilla JavaScript**.  
This project includes a modern UI, smooth animations, and features like marking tasks as DONE, deleting tasks, and adding tasks using the Enter key.

---

## 🌈 Features

### ✔ Add New Tasks  
Quickly add tasks using the input and Add button.

### ✔ Mark Tasks as Done  
A dedicated green **✔ Done** button makes completing tasks satisfying.

### ✔ Delete Tasks  
Remove tasks instantly with the Delete button.

### ✔ Enter Key Support  
Press **Enter** to add tasks faster.

### ✔ Beautiful Animated UI  
- Gradient background  
- Smooth fade & slide animations  
- Rounded task cards  
- Clean playful theme  

---

## 🛠 Technologies Used
- **HTML5**  
- **CSS3 (Modern Styling, Animations, Gradient Backgrounds)**  
- **Vanilla JavaScript (DOM Manipulation, Events)**
